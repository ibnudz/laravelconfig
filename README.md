# laravelconfig
Laravel Retrieving information in .env (such as SMTP, AWS, TWILIO,  SENDGRID, and other.)

# Support On 
Windows, Linux/Unix, Termux

# how to run?
      git clone https://github.com/ibnudz/laravelconfig
      cd laravelconfig
      pip install -r requirements.txt
      # use python3
      python3 larav.py
